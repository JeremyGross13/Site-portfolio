Pedagogical uses

This project has been writed in java, javafx and fxml. Firstly, you can found an home page, were you can see a link for a form online, that I made with HTML and CSS. Then, you have a help button, that show you some help for using our app. Also, you have a connexion button that ask you an user id and a password (the user is "admin" and the password is "password"). The admin can add what they want into our database. After, you have a page were you can choose the view of the uses : either by academies or by theme. And when you choose a view, you have it and you can choose your academy or thematics to view the attached uses.
